# StudentCourseManager

A modern C++ student course management system.

## Features

- Add, remove, update and search students
- Store student data in JSON format
- Command-line menu interface
- Built with CMake
- Written in modern C++17

## Tech Stack

- C++17
- CMake
- nlohmann/json
- VS Code
- WSL / Linux

## Build

```bash
cmake -S . -B build
cmake --build build
